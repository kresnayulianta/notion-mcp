# Contributing

Thanks for your interest in this project!

## Current Status

**This project is not currently accepting code contributions or pull requests.**

I'm building this as a personal tool and want to maintain full control over the direction and implementation for now.

## Issues Are Welcome

If you find a bug or have a question, feel free to [open an issue](../../issues). Please include:

- What you were trying to do
- What happened instead
- Your environment (OS, Node.js version, MCP client)

## Future

I may open contributions in the future. If that changes, this file will be updated.
